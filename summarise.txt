# 🛡️ FlaskGuard WAF - Project Summary

## What is FlaskGuard?

**FlaskGuard** is a lightweight, production-inspired **Web Application Firewall (WAF)** built with Flask that sits between clients and web applications to detect and block malicious requests before they reach the target server.

---

## Core Architecture

```
Client Request → WAF Middleware → Regex Detection → IP Blacklist → Target Server
                                    ↓
                                Block & Log (403)
```

---

## What Has Been Implemented ✅

| Component | Description |
|-----------|-------------|
| **Regex Detection Engine** | 30+ patterns detecting SQLi, XSS, Path Traversal, Command Injection |
| **IP Blacklist Manager** | Auto-bans IPs after 5 strikes; manual ban/unban capability |
| **Attack Logger** | JSON + Python logging with last 500 events stored |
| **File Inspection** | Blocks dangerous file uploads (.exe, .php, .jsp, etc.) |

---

## What Will Be Added 🚀

### 1. ML Classifier
- Self-learning model using Naive Bayes + n-gram TF-IDF
- Catches obfuscated attacks that regex misses
- Active learning pipeline with human feedback loop

### 2. Advanced Rate Limiting
- Token bucket algorithm with sliding windows
- Per-endpoint limits (login: 10/min, search: 200/min)
- Max requests/second control across multiple sites

### 3. Admin Dashboard
- Real-time attack metrics and charts
- Live attack log with filtering
- Blacklist management interface
- ML training interface for false positives

### 4. Multi-Site Proxy
- Single WAF protecting multiple backend servers
- Load balancing and circuit breaker patterns
- Request mirroring for testing
